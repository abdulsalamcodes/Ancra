Quick Start

1. Create an API Key to authenticate your server-side requests.

2. Configure your Nomba credentials under Settings → Nomba Config.

3. Set a Webhook endpoint to receive real-time account events.

4. Use the /accounts API to provision virtual accounts for your customers.
