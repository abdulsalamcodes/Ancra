---
title: Close an Account
---

Closing an account permanently marks it as inactive. Any payments received after closure are routed to a **suspense ledger** and will not credit the customer.

```http
POST /accounts/{id}/close
Authorization: Bearer <key>
```

```json
{
  "status": "closed"
}
```

## Behaviour After Closure

- The account's `status` changes to `closed`
- Inbound payments to the bank account number are captured by Nomba but routed to Ancra's suspense account — **not** credited to the customer
- Outbound transfers can no longer be initiated from a closed account
- Closure is **irreversible** via the API

## Suspense Funds

Funds routed to suspense after closure require manual review and return to sender. Contact support with the Nomba transaction ID to initiate a return.
