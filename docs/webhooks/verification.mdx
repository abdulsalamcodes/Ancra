---
title: Signature Verification
---

Every webhook from Nomba includes an HMAC-SHA256 signature. Ancra verifies this automatically, but if you are forwarding or re-processing events you can verify them yourself.

## Headers

| Header | Description |
|---|---|
| `nomba-signature` | Base64-encoded HMAC-SHA256 digest |
| `nomba-timestamp` | RFC-3339 timestamp of when the event was sent |
| `nomba-signature-algorithm` | Always `HmacSHA256` |
| `nomba-signature-version` | Always `1.0.0` |

> Header names are case-insensitive.

## Signing Algorithm

Nomba does **not** sign the raw body. It signs a colon-separated string of specific payload fields:

```
event_type:request_id:user_id:wallet_id:transaction_id:transaction_type:transaction_time:response_code:timestamp
```

Where:
- `event_type` = `payload.event_type`
- `request_id` = `payload.requestId`
- `user_id` = `payload.data.merchant.userId`
- `wallet_id` = `payload.data.merchant.walletId`
- `transaction_id` = `payload.data.transaction.transactionId`
- `transaction_type` = `payload.data.transaction.type`
- `transaction_time` = `payload.data.transaction.time`
- `response_code` = `payload.data.transaction.responseCode` (use `""` if value is `"null"`)
- `timestamp` = value of `nomba-timestamp` header

## Verification in Go

```go
func verifySignature(body []byte, r *http.Request, secret string) bool {
    sig := r.Header.Get("nomba-signature")
    timestamp := r.Header.Get("nomba-timestamp")

    var p struct {
        EventType string `json:"event_type"`
        RequestID string `json:"requestId"`
        Data struct {
            Merchant struct {
                UserID   string `json:"userId"`
                WalletID string `json:"walletId"`
            } `json:"merchant"`
            Transaction struct {
                TransactionID string `json:"transactionId"`
                Type          string `json:"type"`
                Time          string `json:"time"`
                ResponseCode  string `json:"responseCode"`
            } `json:"transaction"`
        } `json:"data"`
    }
    json.Unmarshal(body, &p)

    rc := p.Data.Transaction.ResponseCode
    if rc == "null" { rc = "" }

    hashStr := fmt.Sprintf("%s:%s:%s:%s:%s:%s:%s:%s:%s",
        p.EventType, p.RequestID,
        p.Data.Merchant.UserID, p.Data.Merchant.WalletID,
        p.Data.Transaction.TransactionID, p.Data.Transaction.Type,
        p.Data.Transaction.Time, rc, timestamp,
    )

    mac := hmac.New(sha256.New, []byte(secret))
    mac.Write([]byte(hashStr))
    expected := base64.StdEncoding.EncodeToString(mac.Sum(nil))

    return hmac.Equal([]byte(expected), []byte(sig))
}
```

## Verification in JavaScript

```javascript
import crypto from "crypto";

function verifySignature(body, headers, secret) {
  const sig = headers["nomba-signature"];
  const timestamp = headers["nomba-timestamp"];
  const p = JSON.parse(body);

  const txn = p.data?.transaction || {};
  const merchant = p.data?.merchant || {};
  let responseCode = txn.responseCode || "";
  if (responseCode === "null") responseCode = "";

  const hashStr = [
    p.event_type, p.requestId,
    merchant.userId, merchant.walletId,
    txn.transactionId, txn.type,
    txn.time, responseCode, timestamp,
  ].join(":");

  const expected = crypto
    .createHmac("sha256", secret)
    .update(hashStr)
    .digest("base64");

  return crypto.timingSafeEqual(
    Buffer.from(expected),
    Buffer.from(sig)
  );
}
```
